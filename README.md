# pointcloud_projection
project pointcloud to fisheye/pinhole by model kb4/omnidirectional/pinhole

the way to run :
1.go to the XXX_projection
2.open terminal in present dirctionary 
3.type cmake .
4.type make
5.type ./XXX_projection
